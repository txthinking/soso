### 修改 Chrome 地址栏的默认搜索引擎为 soso.ooo

1. 在地址栏输入 `chrome://settings/searchEngines`，回车
2. 点击“其他搜索引擎”右侧的**添加**按钮:
    - 搜索引擎: `soso`
    - 关键字: `ss`
    - 网址（用“%s”代替搜索字词）: `https://soso.ooo/search.html?q=%s`
    - 填写完上面三个字段后点击**添加**按钮
3. 然后在 “其他搜索引擎” 列表里找到 `soso`，点击右边的 **三个点图标**，选择 **设为默认搜索引擎**
